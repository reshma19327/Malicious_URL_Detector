# Malicious URL Detection Using Machine Learning

## Overview

This project uses Machine Learning to classify URLs as Safe or Malicious.

## Features Extracted

* URL Length
* Number of Dots
* Number of Digits
* Number of Hyphens
* HTTPS Detection

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Random Forest Classifier
* Git & GitHub
